# 🧠 NLP Tokenization, Stemming, and Lemmatization using NLTK

This project demonstrates various Natural Language Processing (NLP) techniques using the NLTK library in Python. It covers multiple types of tokenization, stemming methods, and lemmatization on a sample sentence.

---

## 🚀 Features

- 🔹 **Tokenization Techniques**
  - Whitespace Tokenization
  - Punctuation-based Tokenization
  - Treebank Tokenization
  - Tweet Tokenization
  - Multi-Word Expression (MWE) Tokenization

- 🔹 **Stemming Techniques**
  - Porter Stemmer
  - Snowball Stemmer

- 🔹 **Lemmatization**
  - WordNet Lemmatizer
